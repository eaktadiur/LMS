Os relatórios existentes são exibidos em forma de listas, com as seguintes opções

para cada relatório: Visualizar, Editar e Excluir.

Para facilitar a localização dos relatórios, o usuário pode fazer uma busca, de acordo

com as condições que são informadas ao sistema.

Durante a visualização de um relatório, o usuário tem as seguintes opções:

Visualizar, Editar, Excluir e Exportar.
